JSON (JavaScript Object Notation) - це текстовий формат обміну даними, який легко читається і записується людьми, а також легко парситься та 
генерується комп'ютерами. В основному, JSON використовується для обміну даними між веб-серверами та клієнтами, але його можна використовувати 
і в інших контекстах.
